# DLRAppStoreRatings

This is a internal CocoaPod.

## Installation

To install it, simply add the following line to your Podfile:

```rb
pod 'DLRAppStoreRatings', :git => 'git@github.com:detroit-labs/dlr-app-store-ratings-ios.git'
```
